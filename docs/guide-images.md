# Guide pour ajouter des images

Ce guide explique comment uploader et utiliser des images sur le site (logo, photos d'actualités…).

---

## Uploader une image

1. Accédez à l'interface d'administration : `https://estebanwendling.github.io/apcm/admin`
2. Dans l'éditeur (présentation ou actualité), repérez le champ **"Image"** ou **"Logo"**
3. Cliquez sur **"Choisir une image"**
4. Cliquez sur **"Upload"** pour envoyer une image depuis votre ordinateur
5. Sélectionnez le fichier image (formats acceptés : JPG, PNG, WebP, GIF)
6. L'image est uploadée et sélectionnée automatiquement

---

## Conseils pour les images

- **Taille recommandée** : moins de 2 Mo par image pour un chargement rapide
- **Format recommandé** : JPG pour les photos, PNG si transparence nécessaire
- **Dimensions pour le logo** : hauteur de 100 à 200 pixels, fond transparent (PNG) de préférence
- **Dimensions pour les actualités** : largeur d'environ 1200 pixels, ratio 16:9 pour un bel affichage

---

## Supprimer une image

Les images uploadées sont stockées dans le dossier `public/images/` du dépôt GitHub. Pour supprimer une image inutilisée :

1. Connectez-vous à GitHub
2. Naviguez vers le dépôt → `public` → `images`
3. Cliquez sur le fichier à supprimer
4. Cliquez sur l'icône corbeille 🗑️
5. Confirmez la suppression

> Attention : vérifiez d'abord que l'image n'est plus utilisée sur le site avant de la supprimer.
