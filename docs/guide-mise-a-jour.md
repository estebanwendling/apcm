# Guide de mise à jour du site APCM

Ce guide explique comment modifier le contenu du site (textes, actualités, coordonnées) sans aucune connaissance technique.

---

## Accéder à l'interface d'administration

1. Ouvrez votre navigateur (Chrome, Firefox, Edge…)
2. Allez à l'adresse : **`https://estebanwendling.github.io/apcm/admin`**
3. Cliquez sur le bouton **"Login with GitHub"**
4. Connectez-vous avec le compte GitHub de l'association
5. Autorisez l'accès si une fenêtre le demande

Vous arrivez sur l'interface d'administration.

---

## Modifier la présentation et les coordonnées

1. Dans le menu de gauche, cliquez sur **"Présentation & Contact"**
2. Cliquez sur le fichier qui s'affiche
3. Vous pouvez modifier :
   - **Nom** et **Slogan** de l'association
   - **Description** : zone de texte enrichi (gras, listes, liens…)
   - **Contact** : e-mail, téléphone, adresse, WhatsApp, liens HelloAsso et Facebook

Pour mettre du texte en gras : sélectionnez le texte, cliquez sur **B** dans la barre d'outils.  
Pour ajouter un lien : sélectionnez le texte, cliquez sur l'icône de lien 🔗, collez l'URL.

---

## Modifier les sections de la page

Les sections sont les blocs de texte qui apparaissent sous la présentation (ex : "Notre mission", "Rejoindre l'association").

1. Toujours dans **"Présentation & Contact"**, faites défiler vers le bas jusqu'à **"Sections"**
2. Cliquez sur une section existante pour la modifier
3. Cliquez sur **"Ajouter une section"** pour en créer une nouvelle
4. Pour supprimer une section, cliquez sur la corbeille 🗑️ à droite

---

## Ajouter ou modifier une actualité

1. Dans le menu de gauche, toujours dans **"Présentation & Contact"**, faites défiler jusqu'à **"Actualités"**
2. Cliquez sur **"Ajouter une actualité"** ou sur une actualité existante
3. Remplissez les champs :
   - **Titre** : titre de l'article
   - **Date** : date de publication
   - **Résumé** : une phrase courte qui apparaît dans la liste
   - **Contenu** : le texte complet (avec mise en forme possible)
   - **Image** *(facultatif)* : voir le guide des images

---

## Publier les modifications

Une fois vos modifications faites :

1. Cliquez sur le bouton **"Publier"** (ou "Save" selon la version)
2. Attendez environ **2 minutes**
3. Rechargez le site public : vos modifications sont en ligne !

> En coulisses, votre clic "Publier" envoie automatiquement les modifications à GitHub, qui recompile et remet en ligne le site. Vous n'avez rien d'autre à faire.

---

## En cas de problème

- Si vous ne pouvez pas vous connecter : vérifiez que votre compte GitHub a bien accès au dépôt de l'association.
- Si le site ne se met pas à jour après 5 minutes : allez sur GitHub → onglet "Actions" pour voir si une erreur s'est produite.
- Pour tout autre problème, contactez la personne qui a mis en place le site.
